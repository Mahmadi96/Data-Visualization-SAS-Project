# Upload Instructions

## Recommended repository settings

- **Repository name:** `netherlands-traffic-accidents-sas`
- **Description:** `SAS Viya Visual Analytics project exploring patterns associated with traffic accidents across the Netherlands.`
- **Visibility:** Public
- Do not initialise the repository with another README or `.gitignore` if uploading this prepared folder.

## Upload using the GitHub website

1. Extract the ZIP file.
2. Open the new GitHub repository.
3. Select **Add file > Upload files**.
4. Drag all files and folders from inside the extracted folder into GitHub.
5. Use the commit message: `Add SAS Visual Analytics traffic accident project`.
6. Commit the upload.
