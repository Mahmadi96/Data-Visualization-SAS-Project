# Data Availability

The source dataset is not included in this repository.

The project used the **Case Study - Dutch Traffic Accidents** dataset available through **SAS Viya for Learners**. Access and redistribution rights may be restricted by the platform or course environment.

To reproduce the dashboard, obtain authorised access to the dataset and use the variable list in the main README and methodology document.

Do not upload restricted course data or personal-level accident records without confirming the relevant licence, privacy requirements and institutional rules.
